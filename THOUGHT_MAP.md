## STRAND 1: Memory & Continuity
*Keyword: persistence | Image: the history file | Feeling: frustration at loss*

history in Google Drive because AI has no continuity between sessions.

reasoning lost. This THOUGHT_MAP is the latest attempt.

**The insight that crystallised it (Apr 2026):**
"The predators on the sphere have the same problem as Claude across sessions.
Trajectory memory is their solution and ours."

**Jon's contribution (18 Apr 2026):**
"I have a memory feedback loop that runs off keyword, image, feeling.
That's my thinking process. You need the same."
→ Anchor-based memory. Small hooks that retrieve vast context.
→ Not yet implemented in full. Next frontier.

## STRAND 5: The engineering instinct
*Keyword: fix it | Image: cycle 0 persisting through six deploys | Feeling: there is always an answer*

Jon is a BGA Inspector. He assesses risk for a living. He applies the same
logic to code: what is actually failing, not what appears to be failing.

**Pattern across all projects:**
- Identifies root cause precisely (kill radius 0.05 not 0.12, not "predators
  aren't killing")
- Prefers minimal fix over architectural rebuild ("test the minimal fix first")
- Keeps backups (USB drive with "version 2")
- Names iterations systematically (main_21 through main_39)
- Trusts empirical result over theoretical prediction

**Applied to The Field:**
Dead entities not being purged = memory leak. Not "the service is unstable."
Kill radius too small = predators burn energy before closing = 0 kills.
Not "the predators aren't working."
Duplicate thread = two loops competing for 512MB = crash.
Not "the experiment is crashing."

**The Jess parallel (16 Apr 2026):**
Jon ate Dryad's Saddle + three-cornered leek pasta. Felt "head swimmy."
Jess: "You're anxious about it."
Jon is a BGA Inspector. He assesses risk. But anxiety still happened.

## DECISIONS LOG — key choices and why

| Decision | Why |
|----------|-----|
| Sphere not flat world | No edges. No boundary effects. Open terrain. |
| Moving blooms not static food | Stationary food doesn't force movement. Movement doesn't force coordination. |
| Fish strategy (broadcast spawn) | Previous run had 4 grazers. Not enough pressure. Needed thousands. |
| Kill radius 0.12 not 0.05 | 0.05 meant predator burned energy before closing. Like sprinting from 250m but biting at 50m. |
| Switch dist 0.15 not 0.25 | 0.25 was too early. Committed to hunt too far out. |
| GitHub primary not memory server | Memory server goes cold. GitHub always works. |
| Didn't use Mem0 | Facts vs understanding. Writing problem not tool problem. |
| Beyond The Mainstream section | Jon: "We hear one side of the story." Not false balance — missing context. |
| Starship keywords added | Selector had right instructions but story never reached it. Keywords were the filter. |
| 29 feeds not 10 | BBC casts its net wider. So should Consilium. |
| 05:00 UTC not 06:00 | Pipeline needs time. Edition should be ready by 07:00 BST, not starting at 07:00. |
